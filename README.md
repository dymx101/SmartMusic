# SmartMusic iOS App

SmartMusic 是一款现代化的音乐播放应用，提供个性化推荐、在线播放、歌单管理等功能。

## 功能特点

- 🎵 在线音乐播放
- 📱 支持后台播放
- 💡 个性化推荐
- 📝 播放列表管理
- ❤️ 收藏功能
- 🕒 播放历史
- 🌙 休眠定时

## 快速开始

### 环境要求

- iOS 15.0+
- Xcode 14.0+
- Swift 5.5+

### 安装步骤

1. 克隆项目
```bash
git clone https://github.com/yourusername/SmartMusic.git
```

2. 打开项目
```bash
cd SmartMusic
open SmartMusic.xcodeproj
```

3. 运行项目
- 选择目标设备或模拟器
- 点击运行按钮或按 Cmd+R

## 项目结构

```
SmartMusic/

├── Views/          # 视图组件
```
